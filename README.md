# Wallpaper
Windows下的动态壁纸软件

## 开源协议
本代码遵守GPLv3.0开源

## 使用
安装requirements.txt内的库即可
```
pip install -r requirements.txt
```
在`config.yml`中的`video`键的值输入视频路径
